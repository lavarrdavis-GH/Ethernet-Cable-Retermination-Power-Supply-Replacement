# Ethernet Cable Retermination & Power Supply Replacement

## Overview
Performed troubleshooting and maintenance on surveillance/network infrastructure by reterminating Ethernet cabling and replacing a faulty power supply box.

## Responsibilities
- Diagnosed connectivity and power issues
- Reterminated Ethernet cable connections using RJ45 connectors
- Replaced defective power supply box with a new unit
- Reconnected power and network cabling
- Verified proper network connectivity and device communication
- Troubleshot post-installation issues to validate successful operation

## Technologies Used
- Ethernet Cabling
- RJ45 Connectors
- Power Supply Units
- Network Infrastructure
- CCTV Systems

## Skills Demonstrated
- Cable Retermination
- Hardware Troubleshooting
- Network Diagnostics
- Infrastructure Maintenance
- Connectivity Testing
